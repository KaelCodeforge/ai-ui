# Design Tokens

AI UI uses CSS variables (prefixed with `--ai-`) as design tokens.

## Brand

- `--ai-color-primary`: `#ff4d4f`
- `--ai-color-primary-hover`: `#ff7875`
- `--ai-color-primary-active`: `#d9363e`

## Semantic colors

- Background: `--ai-color-bg-base`, `--ai-color-fill-default`, ...
- Text: `--ai-color-text-primary`, `--ai-color-text-secondary`, ...
- Border: `--ai-color-border`, `--ai-color-border-strong`
