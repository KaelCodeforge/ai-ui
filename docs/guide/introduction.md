# 介绍

AI UI 是一个基于 Vue 3 的组件库，专为中后台应用场景设计。

## 特性

- **Vue 3 + TypeScript** - 完全基于 Vue 3 Composition API 和 TypeScript 开发
- **完整类型支持** - 所有组件都有完整的 TypeScript 类型定义
- **主题定制** - 基于 CSS 变量的主题系统，支持亮色/暗色模式
- **AI Coding 友好** - 统一的代码结构和命名规范，便于 AI 辅助开发

## 设计原则

### 一致性

所有组件遵循统一的设计语言和交互模式，确保用户体验的一致性。

### 可访问性

组件设计考虑无障碍访问需求，支持键盘导航和屏幕阅读器。

### 可定制性

通过 CSS 变量和 props，可以轻松定制组件的外观和行为。

## 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue | 3.4+ | 核心框架 |
| TypeScript | 5.0+ | 类型系统 |
| Vite | 5.0+ | 构建工具 |
| Sass | 1.69+ | 样式预处理 |

## 浏览器支持

AI UI 支持所有现代浏览器：

- Chrome >= 87
- Firefox >= 78
- Safari >= 14
- Edge >= 88
